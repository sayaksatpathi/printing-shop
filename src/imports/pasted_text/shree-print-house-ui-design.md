Design a sharp, professional, and creative printing business website UI 
for "Shree Print House" located on Boring Road, Patna, Bihar.

Brand Identity:
- Tagline: "Print It Right. Every Time."
- Sub-tagline: "Boring Road's Most Trusted Printing Partner Since 2005"
- Services: Offset Printing, Digital Printing, Visiting Cards, Flex/Banners, Brochures, Packaging, Stationery
- Voice: Bold, reliable, professional, creative studio energy

Style Direction:
- Bold editorial-industrial aesthetic (not corporate boring)
- Clean white + deep black base with strong ink-red accents
- Add subtle textures: CMYK halftone dots, paper grain, ink streaks
- Asymmetric layouts with tight spacing and high contrast
- Feels like a premium modern print studio

Color System:
- White: #FFFFFF
- Black: #111111 / #0D0D0D
- Primary Red: #D72638
- Amber: #F5A623
- Cyan accent: #0097A7
- Section tone: #F8F8F6
- Borders: #E0DDD8

Typography:
- Headings: Bebas Neue (ALL CAPS, bold, wide)
- Subheadings: Montserrat (ExtraBold / SemiBold)
- Body: Source Sans Pro (clean and readable)
- Labels: Montserrat caps with spacing

Important:
- Automatically include realistic sample images (printing products, brochures, flex banners, packaging, visiting cards)
- Every component must be clickable and prototype-ready (buttons, cards, gallery, nav links)
- Design full responsive layouts (desktop, tablet, mobile)
