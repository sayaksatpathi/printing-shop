
  # Generate content

  This is a code bundle for Generate content. The original project is available at https://www.figma.com/design/SpGQK7xocciGuWzSGgOfXD/Generate-content.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deploying to Vercel

  1. Go to Vercel → **Add New...** → **Project**
  2. Import the GitHub repo: `sayaksatpathi/printing-shop`
  3. Framework preset: **Vite**
  4. Build command: `npm run build`
  5. Output directory: `dist`
